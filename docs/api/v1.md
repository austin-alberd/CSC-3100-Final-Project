# API Spec V1

## Information Routes

### POST /api/skills (credentials, experience)
  - Accepts: `skill` and `description` as body parameters
  - Returns: `status`, `recordID`

### PUT /api/skills (credentials, experience)
  - Accepts: `skill` and `description` as body parameters
  - Returns: `status`, `recordID`

### DELETE /api/skills (credentials, experience)
  - Accepts: `recordID` as a URL parameter
  - Returns: `status`, `recordID`

### GET /api/skills (credentials, experience)
  - Accepts: nothing
  - Returns: An array of JSON objects `rows`
